# Turkish Translation Guidelines

## Core Rule

**Keep globally recognized technical terms in English.** Translate headers and body text to Turkish, but preserve technical vocabulary.

---

## Terms to Keep in English

| English Term | Incorrect Turkish | Correct Usage |
|--------------|-------------------|---------------|
| Deploy | Dağıtım yapmak | Deploy etmek |
| Pipeline | Boru hattı | Pipeline |
| Commit | İşlemek / Taahhüt | Commit |
| Push | İtmek | Push |
| Pull | Çekmek | Pull |
| Endpoint | Uç noktası | Endpoint |
| Merge Request | Birleştirme isteği | Merge Request |
| Bugfix | Hata düzeltme | Bugfix |
| Feature Flag | Özellik bayrağı | Feature Flag |
| Rollback | Geri alma | Rollback |
| Cache | Önbellek | Cache |
| Timeout | Zaman aşımı | Timeout |
| Retry | Yeniden deneme | Retry |

---

## Terms to Translate

| English | Turkish |
|---------|---------|
| Production | Canlı ortam |
| Non-Production | Canlı dışı ortam |
| Business Requirements | İş Gereksinimleri |
| Technical Implementation | Teknik Implementasyon |
| Problem Statement | Problem Tanımı |
| Proposed Solution | Önerilen Çözüm |
| Definition of Done | Tamamlanma Tanımı |
| Risks | Riskler |
| Test Scenarios | Test Senaryoları |

---

## Examples

### Bad Example
```
Birleştirme isteği gönderildi. Dağıtım boru hattı başlatıldı.
```

### Good Example
```
Merge Request gönderildi. Deploy Pipeline başlatıldı.
```
